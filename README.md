# Human Development Index (HDI) Prediction

## Project Description
This project predicts the Human Development Index (HDI) category of a country using Machine Learning. The model analyzes important development indicators such as Life Expectancy, Mean Years of Schooling, Expected Years of Schooling, and Gross National Income (GNI) per capita to classify a country into one of the HDI categories:
- Very High
- High
- Medium
- Low
